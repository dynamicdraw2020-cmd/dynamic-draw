Dynamic Draw — 가장 먼저 읽어주세요
=====================================

이 ZIP 안에는 실제 홈페이지 소스코드, Supabase DB 설치 SQL, 사용자·관리자 설명서가 모두 들어 있습니다.

딱 4가지만 먼저 하세요.

1) ZIP 파일에 마우스 오른쪽 버튼 → "모두 압축 풀기"
2) 압축 푼 폴더의 "00_여기부터_더블클릭.html"을 두 번 누르기
3) 같은 폴더의 "1_SUPABASE에_한번만_붙여넣기.sql"을 메모장으로 열어 전체 복사
4) Supabase → SQL Editor → New query → 붙여넣기 → Run

중요
----
- SQL 파일은 Supabase 화면 안에서 찾는 것이 아닙니다.
- 지금 다운로드한 ZIP을 압축 푼 폴더 안에 있습니다.
- GitHub에는 ZIP 파일 한 개가 아니라, 압축 푼 폴더 안의 파일과 폴더를 올립니다.
- .env, Secret key, 비밀번호는 GitHub에 올리지 마세요.
