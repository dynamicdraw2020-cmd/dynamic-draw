DynamicD v1.6.2 카테고리 펼치기/접기 복구 핫픽스

목표
- v1.6.1에서 관리자 사이드바 카테고리 UI가 기존 CSS 구조와 어긋난 문제를 복구합니다.
- 관리자 메뉴 카테고리별 접기/펼치기 기능을 다시 사용할 수 있게 합니다.
- '관리자 가입 게이트', 'Signup Gate', 'CS Button', 'Issued Codes' 같은 임시/영문/이상한 라벨을 제거합니다.
- v1.6.1에서 추가한 가입 시크릿코드, CS_MANAGER, 서버 상태, 트래픽 체크 기능은 유지합니다.

교체 방법
1. 이 압축 파일을 풀어 GitHub 저장소 루트에 덮어쓰기합니다.
2. Vercel 자동 배포가 Ready 상태인지 확인합니다.
3. 관리자 페이지에서 새로고침 후 왼쪽 메뉴 카테고리의 접기/펼치기를 확인합니다.

중요
- app/globals.css는 건드리지 않았습니다.
- DB 스키마 변경은 없습니다.
- SQL 파일은 적용 기록용 마커만 포함합니다. 실행은 선택입니다.
