Dynamic D 다음 업데이트 추천 기능 메모
작성일: 2026-06-27

1. 이벤트별 전용 상세 페이지
- /events/[slug] 형태로 이벤트별 안내, 기간, 참여 방법, 주의사항을 따로 보여주기.
- 공지와 추첨/교환 규칙을 한 페이지에 묶으면 유저가 덜 헷갈림.

2. QR 회원 확인
- 유저의 고유 회원 ID를 QR로 표시.
- 관리자가 현장에서 QR을 스캔해서 추첨권/화폐/상품 지급.

3. 지급 내역 CSV 다운로드
- 추첨권 지급, 화폐 지급, 교환, 전체 추첨 결과를 CSV로 내려받기.
- 이벤트 종료 후 운영 기록 정리에 좋음.

4. 일괄 회원 승인
- 승인 대기 회원을 체크박스로 여러 명 선택해서 한 번에 승인.
- 이벤트 직전 가입자가 몰릴 때 필요.

5. 테스트 모드
- 실제 결과/지급 기록을 남기지 않고 관리자만 추첨 흐름을 테스트.
- 라이브 이벤트 시작 전 점검용.

6. 이벤트별 관리자 권한
- 운영자 A는 이벤트 1만, 운영자 B는 이벤트 2만 관리.
- 운영자가 늘어나면 안전하게 권한을 나누는 데 필요.

7. 추첨 결과 공개 이미지 생성
- 당첨 결과를 이미지 카드로 자동 생성.
- 디스코드/오픈채팅에 공지하기 쉬움.

8. 공지 상단 배너
- 긴급 공지나 진행 중 이벤트를 홈페이지 최상단 배너로 표시.
- 서버 점검, 보상 지급, 이벤트 마감 안내에 유용.

9. 유저별 활동 로그
- 유저가 받은 추첨권, 사용한 추첨권, 교환한 내역을 한 화면에서 확인.
- 문의 대응이 쉬워짐.

10. 전체 추첨 재추첨 정책
- 당첨자가 연락 불가일 때 재추첨 가능 여부와 사유 기록.
- 재추첨 버튼은 최고 관리자만 가능하게 제한 추천.
