# 일반 회원 사용 설명서

## 1. 홈페이지 보기

`https://dynamic2020.com`에 들어가면 로그인하지 않아도 진행 중인 뽑기, 상품 확률, 실시간 연출, 최근 결과, 누적 통계를 볼 수 있다. 관리자가 공개 통계를 꺼 두었다면 통계 일부는 보이지 않을 수 있다.

## 2. 회원가입 신청

1. 오른쪽 위 **로그인**을 누른다.
2. 아래의 **회원가입 신청**을 누른다.
3. 이름 또는 닉네임, 연락처, 이메일, 비밀번호를 입력한다.
4. **가입 신청**을 누른다.
5. 이메일 확인 기능이 켜져 있으면 받은 편지함에서 인증 링크를 누른다.
6. 관리자가 확인할 때까지 기다린다.

가입 신청만으로 교환 기능이 열리지는 않는다. 관리자가 승인하면 서버가 중복되지 않는 고유 ID를 자동 발급해야 한다.

## 3. 승인 뒤 로그인

1. **로그인** 페이지에서 이메일과 비밀번호를 입력한다.
2. 승인 전이면 “관리자 승인 대기 중” 화면이 나온다.
3. 승인 뒤에는 **내 정보** 화면으로 이동한다.
4. 관리자가 신청을 반려하거나 계정을 정지했다면 안내된 사유를 확인하고 운영자에게 문의한다.

## 4. 비밀번호를 잊었을 때

1. 로그인 화면에서 **비밀번호를 잊었나요?**를 누른다.
2. 가입에 사용한 이메일을 입력하고 복구 메일을 요청한다.
3. 받은 메일 안의 링크를 누른다.
4. 새 비밀번호를 두 번 입력하고 저장한다.
5. 새 비밀번호로 로그인한다.

메일이 보이지 않으면 스팸함을 확인한다. 여러 번 눌러도 가장 최근에 받은 링크를 사용한다.

## 5. 고유 ID 확인

내 정보 상단에 `DD-2026-000001` 같은 번호가 표시된다. 이 번호는 회원의 추첨 결과, 보유 상품, 교환 내역을 연결하는 운영 번호다. 비밀번호는 아니지만 다른 사람의 번호를 자신의 번호처럼 사용하지 않는다.

## 6. 실시간 추첨 보기

1. 상단의 **실시간 결과**를 누른다.
2. 관리자가 추첨을 누르면 카드가 흔들린다.
3. 빛 효과가 나타난다.
4. 카드가 뒤집힌다.
5. 상품명과 일부 가려진 회원 정보가 나타난다.

화면을 새로고침하지 않아도 된다. 중간에 접속하거나 새로고침해도 최근 진행 이벤트를 확인해 연출 또는 결과 상태를 복구한다. 결과는 연출 전에 서버와 DB에서 이미 결정되며 브라우저가 상품을 정하지 않는다.

## 7. 보유 상품 보기

**내 정보**에서 상품명과 수량을 확인한다. `꽝`처럼 보관하지 않는 상품은 보유함에 들어오지 않는다. `찢어진 입장권`, `DwX`, `Dynamic`처럼 관리자가 보관 상품으로 설정한 항목만 수량에 반영된다.

보관 상품은 카드 결과가 실제로 공개되는 순간 한 번만 적립된다. 연출 중에는 아직 보유함에 보이지 않을 수 있다.

## 8. 상품 교환하기

1. 내 정보에서 **교환 탭으로 이동**을 누른다.
2. 현재 보유한 재료 수량을 확인한다.
3. 가능한 규칙의 **교환하기** 버튼을 누른다.
4. 확인 창에서 승인한다.
5. 재료가 자동 차감되고 새 상품이 자동 지급된다.

예: 찢어진 입장권 5개를 DwX 1개로 교환한다. 수량이 부족하면 버튼이 비활성화된다. 인터넷이 느려 버튼을 두 번 눌러도 같은 요청은 한 번만 처리되도록 설계되어 있다.

## 9. 행사장에서 관리자에게 교환 요청하기

회원이 직접 로그인하기 어려운 행사장에서는 고유 ID를 관리자에게 보여 줄 수 있다. 관리자는 **교환 시스템 → 관리자 현장 교환**에서 고유 ID와 교환 규칙을 확인하고 대신 처리한다. 재료 차감과 대상 상품 지급은 회원이 직접 교환할 때와 똑같이 한 번의 DB 작업으로 처리되고 관리자 로그에도 남는다.

비밀번호는 관리자에게 알려 주지 않는다. 현장 교환에는 고유 ID만 사용한다.

## 10. 내 기록 확인

내 정보에서 다음 항목을 확인할 수 있다.

- 최근 본인 추첨 결과
- 당첨 시각과 상품
- 현재 보유 상품 수량
- 교환한 재료·수량과 받은 상품·수량

공개 페이지에는 이름과 고유 ID가 일부 가려져 표시되지만, 내 정보에서는 본인 기록을 확인할 수 있다.

## 11. 로그아웃

공용 컴퓨터나 행사장 태블릿을 사용했다면 작업이 끝난 뒤 반드시 **로그아웃**을 누른다. 브라우저 창만 닫는 것보다 로그아웃이 안전하다.

## 12. 문제가 있을 때

- **로그인이 안 됨:** 이메일·비밀번호, 이메일 인증 여부를 확인한다.
- **복구 메일이 안 옴:** 스팸함을 보고, 가입 이메일 철자를 다시 확인한다.
- **승인 대기만 나옴:** 관리자가 아직 승인하지 않은 상태다.
- **교환 버튼이 안 눌림:** 재료 수량이 부족하거나 규칙이 꺼져 있다.
- **결과가 늦게 나옴:** 연출 시간 3~5초 뒤 자동 공개된다. 인터넷이 끊겼다가 돌아오면 복구 기능이 공개 상태를 다시 확인한다.
- **내 수량이 이상함:** 관리자에게 고유 ID와 발생 시각을 알려 준다. 관리자는 추첨·교환·감사 기록을 확인할 수 있다.
- **계정이 정지됨:** 정지 사유를 확인하고 행사 운영자에게 문의한다.
